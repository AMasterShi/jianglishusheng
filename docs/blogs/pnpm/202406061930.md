## 通过 npm 安装 pnpm

```text
npm install -g pnpm
```
或者
```text
npm install -g @pnpm/exe
```

## 兼容性

以下列表列出了以往的 pnpm 版本和对应支持的 Node.js 版本

| Node.js    | pnpm 7 | pnpm 8 | pnpm 9 |
| ---------- | ------ | ------ | ------ |
| Node.js 12 | ❌    | ❌     | ❌    |
| Node.js 14 | ✔️    | ❌     | ❌    |
| Node.js 16 | ✔️    | ✔️     | ❌    |
| Node.js 18 | ✔️    | ✔️     | ✔️    |
